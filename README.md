# LaTeX Workspace Template
Requires the use of the following VS Code extensions:

* [LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop)
* [LaTeX+](https://marketplace.visualstudio.com/items?itemName=ltex-plus.vscode-ltex-plus)
* [Paste Image](https://marketplace.visualstudio.com/items?itemName=mushan.vscode-paste-image)

Settings for the extensions are included in <code>./vscode/settings.json</code><br>

For further reading, [Paul Wintz](https://paulwintz.com/latex-in-vscode/) has a great article about VS Code configuration for LaTeX documents.
